template<size_t N>

VarArray(&)  outData[N],
long long int  timeStep = 0 
)
inline

read the data from the file at the given filepath and parse all given variables. the outData array contains all the needed information (array, type, length, stride) to work with the data. If the timeStep does not matter or should be ignored set it to zero.

IMPORTANT the size of variableName and outData should match N

Template Parameters
Nnumber of variables to parse
Parameters
filepaththe path to the netCDF file
variableNamethe names of the variables to parse
outDatathe parsed data for each variable with the type, length and stride. If the VarArray is destroyed the read data 'VarArray::array' is destroyed to.
timeStepOPTIONAL start of data parsing. The timeStep uses the dimension directly therefore index values are required e.g. 0, 1, 2, ... . Can also be negativ to read from the back.
